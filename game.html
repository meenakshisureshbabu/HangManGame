THE VANISHMAN GAME

  • This is a two player game Player1 Vs Computer
  • Press "Start Play" button to start the game
  • Guess the word using the clue given and start typing a single letter at once
  • Each player have 3 guesses
  • Whoever found the word would be the winner in that round and they will be scored.
  • When you want to quit the game, the winner of this game will be decided based on the scores.
  • You can play any number of words.
  • Always press Restart whenever you want to restart the new game.
